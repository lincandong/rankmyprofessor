# InfoProf子系统需求

优先级为0～10，10表示最先开发，1表示最后开发，0表示不开发。

## 基本部分

- 老师信息展示页面（这肯定得有）：10
- 分类打分（即，学生不给出一个总评分而是根据“作业难度”、“授课质量”等指标分类计算，我们自己写算法计算出总分）：5
- 万能搜索框：10
- 万能搜索框悬浮窗（在页面上有一个按钮，类似于回到顶部，但是起到搜索框的功能）：7
- 学生对老师/课程的评论/评分：8
